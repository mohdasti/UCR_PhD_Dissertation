# Chapter 1 Figures

This directory contains figures for Chapter 1 (LCYA behavioral manuscript).

## Required Figures

You need to copy the following figures from your LCYA repository:

1. **figure1_task_schematics.png** - Task schematics showing CDT, ADT, and VDT trial structure
   - Source: `LCYA/figures/publication/Figure1_Task_Schematics.png` (or similar)
   
2. **figure2_performance.png** - Behavioral performance (accuracy and RT) across tasks
   - Source: `LCYA/figures/publication/Figure2_Task_Performance_Effects.png`
   
3. **figure3_pupil_waveforms.png** - Pupil response waveforms
   - Source: `LCYA/figures/publication/Figure3_Pupil_Waveforms.png`
   
4. **figure4_auc_plots.png** - AUC bar plots (Total and Cognitive AUC)
   - Source: `LCYA/figures/publication/Figure4_Merged_Total_Cognitive_AUC.png`

## Instructions

1. Navigate to your LCYA repository: `~/Documents/GitHub/LCYA/figures/publication/`
2. Copy the PNG files to this directory
3. Rename them to match the filenames above (lowercase, underscores)

Alternatively, you can create symbolic links:

```bash
cd ~/Documents/GitHub/UCR_PhD_Dissertation/figures/chapter1
ln -s ~/path/to/LCYA/figures/publication/Figure1.png figure1_task_schematics.png
# ... repeat for other figures
```
